## Modern Compiler Implementation in Java (2nd. Ed.) 실습 자료

 - 이 저장소는 Andrew W. Appel 교수의 *Modern Compiler Implementation in Java* (2nd Ed.) 실습 자료와, Jens Palsberg 교수가 이 교재를 바탕으로 진행한 컴파일러 수업의 실습 자료를 모아 정리한 것입니다.
교재에서 안내된 원래의 실습 자료 사이트 링크가 더 이상 유효하지 않아, 이 저장소를 통해 관련 자료를 보존하고 참고할 수 있도록 하였습니다.

 - 전남대학교 인공지능학부/소프트웨어공학과 컴파일러 과목의 실습 자료로 활용하고 있습니다.
